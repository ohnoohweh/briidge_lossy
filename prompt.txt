1) UDP Frame Bridge/Proxy
   - Listens on UDP :443.
   - Connected UDP client to 127.0.0.1:16666.
   - Parses fixed 1200-byte frames with this structure:
       MAGIC (20 bytes): C8 00 00 00 02 08 01 02 03 04 05 06 07 05 63 5F 63 69 64 00
       LEN (2 bytes): xx yy, where following_len = (xx*256 + yy) - 0x4000
       RESERVED (1 byte): 00
       COUNTER (2 bytes): package counter (big endian)
       TYPE (1 byte):
         0x00 = single message
         0x01 = first part of multi-frame message
         0x02 = consecutive part
         0x03 = keep_alive message (after 100ms of inactivity)
         0x04 = restart from package counter

       If TYPE==0x01: OVERALL_LEN (2 bytes)
       if TYPE==0x04: package counter to restart (2bytes)
       PAYLOAD
       PADDING: zeros to 1200 bytes
   - Validates header, length encoding, reserved byte, padding, and sequence continuity (warn on issues).
   - Assembles multi-part messages and forwards to 127.0.0.1:16666.
   - In reverse, receives payloads from 127.0.0.1:16666, fragments (MTU=1200 frame size rules) and sends back to the last upstream sender on :443.
   
2) UDP Server (127.0.0.1:16666)
   - On receiving data: prints CRC32 (uppercase hex 8 chars), payload length, and HEX payload (uppercase, no spaces).
   - On ENTER key press: generates random payload (1..1600 bytes), prints same info, and sends it to 127.0.0.1:16666 (loopback).

3) UDP Client (connected to 127.0.0.1:16666)
   - Binds to ephemeral local port, connects to 127.0.0.1:16666.
   - On receiving data: prints CRC32, length, and HEX.
   - On ENTER key press: generates random payload (1..1600 bytes), prints same info, and sends it to the server.

4) Dependencies for all scripts
   - Only Python standard library modules were used (no pip install needed):
     asyncio, socket, argparse, sys, time, typing, zlib, secrets.
