# Testing docs

Detailed testing notes are in `docs/README_TESTING.md`.

This now includes dedicated sections for:
- `tests/integration/test_overlay_e2e.py`
- `tests/integration/test_overlay_e2e.py --mode reconnect` (optional override)

- available command-line options,
- pytest mode support (including marker filters),

- Pytest execution mode is supported; see `docs/README_TESTING.md` for copy/paste commands.
